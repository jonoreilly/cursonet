<figure> y <figcaption>

<figure> no solo sirve para insertar imágenes


Se utiliza para insertar elementos que tienen algo que ver con el contenido pero, a su vez, son elementos independientes. Las imágenes y los vídeos cumplen con esa condición.


<figcaption> nos permite poner un título a pie de imagen.


El sentido de utilizar <figure> o <figcaption> no es el de construir la estructura de la página web sino la de darle un "sentido semántico" para que cualquier dispositivo sepa cual es el contenido de esas etiquetas.


Otras etiquetas semánticas

Etiqueta Uso semántico en html5
<mark></mark> Marcar un texto sin importancia ni énfasis (pe: resultado de una búsqueda).
<em></em> Para indicar énfasis. Sustituye a la etiqueta <i>.
<strong></strong> Marcado de texto importante. Sustituye a la etiqueta <b>.
<small></small> Para marcar textos legales.
<cite></cite> Títulos de películas, canciones, trabajos propios...
<address></address> Para información de contacto.
<time></time> Fechas y horas.

Este texto nos va a servir como ejemplo de las etiquetas semánticas del cuadro adjunto. Ésta, por ejemplo, es una etiqueta <strong>.


Las etiquetas <em> sirven para indicar énfasis.El texto aparece en cursiva


Con <mark> resaltamos texto que no es importante.Produce el efecto de marcado del típico rotulador amarillo fosforito.


Hay que resaltar la importancia del sentido semántico de estas etiquetas más que su sentido visual. Es importante comprender la diferencia.


El texto que está en las etiquetas <strong> es bastante importante para los buscadores a la hora de posicionar las paginas web y categorizarlas. Si es importante para ti, también lo es para el buscador de turno.


La etiqueta <small> se utiliza para presentar textos legales, copyright etc. Si le damos otro uso, como en este caso, no estaremos respetando al 100% el standard.


Voy a nombrar un disco de The Beatles para utilizar la etiqueta <cite>:Rubber Soul es probablemente su mejor disco.


Con <address> pongo una dirección o teléfono. También produce un salto de carro automático ya que esta etiqueta es de tipo block.

Tlfno: 333 333 333 333


La etiqueta <time> representa horas y fechas. Tiene sus propios atributos como datetime (donde introducimos la fecha y la hora) o pubdate que indicará que la fecha de publicación ha sido la indicada en datetime.
